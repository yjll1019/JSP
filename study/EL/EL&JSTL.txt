1. 표현 언어(EL : Expression Language)
 - 스크립트릿에서 자바 코드를 사용의 불편함을 해소하기 위해 사용
 - ${} 이런 형태로 사용 
 - ${} 이 형태를 그대로 출력하고 싶다면 \${} << 이렇게 작성할 것
 - JSTL 조건식과 함께 사용된다.
 - 내장 객체 param과 paramValues를 통해서 요청 파라미터를 처리
 - 값이 null인 경우 null이 아닌 공백으로 표기.
 - 표현언어에서의 ==연산자 : 객체에 저장된 값(내용)을 비교 (equals메소드 역할)

2.JSTL(JSP Standard Tag Library)
 - JSP에서 사용 가능한 표준 태그 라이브러리
    라이브러리 : 여러 프로그램이 공통으로 사용하는 코드를 모아놓은 코드의 집합
 - 코드가 깔끔해지고 가독성이 좋아짐
 - 커스텀 태그 : 기본적으로 제공되는 것이 아닌 자신이 추가한 태그 
 - 커스텀 태그 라이브러리 : 작성한 커스텀 태그를 압축한 후 배포해서 사용하는 것
